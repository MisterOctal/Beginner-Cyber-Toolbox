# Beginner-Cyber-Toolbox
Foundational cybersecurity tools built during my HND and Security+ studies as I explore the basics of networking, sockets, and ethical hacking tools.

## 🚀 Included Scripts
### 1. Port Scanner 
* **Purpose:** Identifies open TCP ports on a target IP.
* **Concepts:** TCP Handshake, Scanning using `socket`, Input Validation.

### 2. Banner Grabber
* **Purpose:** Enhanced version of the port scanner that also retrieves service identification strings (banners) from open ports to identify software versions and OS details.
* **Concepts:** Service Enumeration, Data Transmission using `recv`, String Decoding.

### 3. Password Complexity Evaluator
* **Purpose:** Evaluates the strength of a password against common security policies and provides actionable improvement suggestions.
* **Concepts:** Security Policy Enforcement, Pattern Matching using `re` (Regular Expressions), User Feedback Loops.
